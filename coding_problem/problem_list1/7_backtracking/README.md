# 백트래킹 & 백트래킹 알고리즘

## 백트래킹(backtracking)
어떤 가능성이 없는 곳을 알아보고 되돌아 가는 것

## 백트래킹 알고리즘
가능성이 없는 곳에서는 되돌아가고, 가능성이 있는 곳을 탐색하는 알고리즘

## 유망 함수
'해가 될 가능성을 판단하는 것'이 백트래킹 알고리즘의 핵심

**진행 과정**  
1. 유효한 해의 집합을 정의한다.
2. 위 단계에서 정의한 집합을 그래프로 표현한다.
3. 유망 함수를 정의한다.
4. 백트래킹 알고리즘을 활용해서 해를 찾는다.
